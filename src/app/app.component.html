About the Postcoordination demonstrator

This demonstration complements SNOMED International's Practical Guide to Postcoordination (Currently under development)

This front end application demonstrates possible implementations of clinical user interfaces that support postcoordination and offers a technical view to the process of the generation of postcoordinated expressions in Close to USer Form, how these expressions are submitted to an expression repository, how the Classifiable Form is created following any supported transformation patterns, and how the Necessary Normal Form is generated after running a description logic classifier.

The expressions generated and stored in the expression repository are available for queries using ECL.

The expression processing and storage is performed by a development version of Snowstorm (Snowstorm-X) that implements the necessary FHIR APIs as described in the postcoordination guide.
